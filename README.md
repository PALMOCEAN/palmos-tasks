# Palmo's Tasks (Drift)

**🔗 Живое демо: [palmocean.github.io/palmos-tasks](https://palmocean.github.io/palmos-tasks/)**

Минималистичный менеджер задач: задачи пишутся прямо в заметках markdown-подобным синтаксисом, а затем автоматически собираются в доску и фокус-таймер. Работает полностью в браузере — без сервера и регистрации.

## Возможности

- **Заметки** — свободный текст, задачи задаются строкой вида `- [ ] Текст #must ~2h`.
- **Доска** — все задачи из заметок автоматически группируются по приоритету (MoSCoW: must / should / could).
- **Фокус-режим** — таймер на выбранную задачу с сохранением состояния между перезагрузками.
- **Проекты** — группировка заметок по проектам.
- **Локальное хранение** — все данные в `localStorage`, ничего не уходит на сервер.

## Технологии

- **Next.js 16** (App Router, React 19)
- **Tailwind CSS v4**, shadcn/ui, lucide-react
- **TypeScript**
- Хранение состояния — `localStorage` (без бэкенда)

## Запуск

```bash
pnpm install
pnpm dev
# http://localhost:3000
```

## Синтаксис задач

Внутри любой заметки:

```
- [ ] Сверстать лендинг #must ~2h
- [ ] Написать тесты #should ~1h
- [x] Выполненная задача #could ~30m
```

- `#must` / `#should` / `#could` — приоритет
- `~2h`, `~30m` — оценка времени
- `[x]` — отметка о выполнении

## Архитектура

